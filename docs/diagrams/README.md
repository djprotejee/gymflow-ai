# Діаграми GymFlow AI

Ця директорія містить технічні діаграми для архітектурної та експериментальної частини проєкту.

Файли:

- `c4-context.mmd` – контекст системи;
- `c4-container.mmd` – container-рівень архітектури;
- `er-core-data-model.mmd` – ER-структура основних таблиць застосунку;
- `data-preparation-pipeline.mmd` – pipeline підготовки даних;
- `forecasting-experiment-pipeline.mmd` – pipeline експериментів і вибору моделі;
- `assistant-rag-pipeline.mmd` – flow AI Coach та retrieval-компонента.
